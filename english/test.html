# 前言

TIP

本人菜鸟, 很差的菜鸟, 记录自己所学路程, 或许不适合你, 我会写出我的方法, 也会发出优秀链接, 希望能够帮到你

# 路程

  • 把自己手机调成英文模式, 不会的自己单词查, 养成习惯, 导致现在看不懂的就查
    • 网易有道单词查单词, 设置查词收藏, 抽空复习收藏单词
    • 百度翻译 查句子(因为查句子下面单词也会标记出来, 还可以发音翻译, 网易有道单词句子不会而且遇到两次翻译错了)
  • APP
    • 英语流利说 - 强烈推荐, 但不推荐初学者, 也不推荐加班比较多者
    • 扇贝 - 背单词和句子 (适合考试, 有单词书, 但可记忆性不强, 不要忽视单词下面的句子)
    • 百词斩 && 沪江开心词场 - 朋友推荐 但没怎么用过, 优势有场景
    • 省心英语(NCE) - 开源免费集各一身 每天用扇贝背单词, 不要图快, 也不要图一定背过, 先加深记忆
