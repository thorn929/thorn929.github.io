# 阅读

TIP

  1. 人微言轻, 本文只是劝诫, 时刻提醒自己罢了
  2. 请记得尽管可能失去一切：工作、财产、金钱甚至你的健康，但是你永远不会失去知识。
  • 常言道, 书翻得多了, 脸就翻得少了
  • 每当遇到喜欢的书就总喜欢买下来, 然而读过得寥寥无几, 只能慢慢看, 整理出来一个书单, 稍后我会发出来, 希望能帮到你, 如果你也有的话希望在下方评论推荐给我, 不仅仅技术类型书籍, 非常感谢
  • 无论是技术方向的书，还是其他，只要是对我们个人成长有益的，都应该读, 这应该让我们养成习惯

# 读书的好处

  • 获取知识, 增见识长学问, 拓展思路, 改变思维习惯
    • 读的每本书都会充实着你自己, 你拥有的知识越多，你的装备就越好，你应对挑战的能力就越强, 记得老师说的一句话, 用珠宝装饰自己不如用书籍才充实自己
    • 为什么读书, 记得有个回答, 他说 书籍已经化为我身体的一部分, 就像我曾经吃过的饭, 喝过的水, 早已经化为我的血肉, 但当它需要的时候, 会随时迸发出来
    • 培根说：「读书足以怡静、足以博采、足以长才。」
  • 修养气质
    • 例: 董卿
  • 刺激大脑神经的发展, 使你的大脑不会退化, 增强思考能力

# 推荐

  • 娱乐至死 人类有追逐娱乐的天性，而人类也有保持理性的良知
    • 例: 抖音
